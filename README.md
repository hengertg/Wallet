# 💳 Wallet System

A Python project built to practice **Object-Oriented Programming (OOP)** and simulate the core features of a digital wallet and banking system.

## 🚀 Features

- Deposit money
- Withdraw money
- Savings Account
- Annual Interest Rate
- Balance management
- Account inheritance

## 🧠 Concepts Practiced

- Classes and Objects
- Inheritance
- Encapsulation
- Method overriding
- Object state management
- Basic banking logic

## 🛠 Technologies

- Python 3

## 🎯 Purpose

This project was built as part of my journey to become a Software Engineer. Instead of only learning syntax, I'm focusing on understanding the logic behind software design and Object-Oriented Programming by building real projects.
